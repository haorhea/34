<p>空条承太郎</p> 
<p>.</p> 
<p>空条承太郎的单恋伊始于二十一岁。</p> 
<p>其实要说感情的萌芽究竟是什么时候探头的，他自己也说不太清楚，从相识到真正明确意识到“我喜欢上了花京院”之间的四年时间中，他们的关系太好，走得也太近，因此任何一个阶段的心情似乎都跟薛定谔的猫一样，时刻处于单恋和普通友情的叠加态。</p> 
<p>埃及一战虽然凯旋而归，然而留下的伤痕和后遗症却不会因为战斗的胜利而消失，高中的后两年中承太郎都陪着花京院在SPW的各个下属医院之间奔走，调理他那光是能活下来就能称之为奇迹的身体，而更为奇迹的是，仅仅两年的治疗和复健之后，花京院的身体竟然也恢复得七七八八，差不多和常人无异了。</p> 
<p>承太郎对此自然非常高兴，尽管他完全不介意像刚从埃及回来的时候一样为花京院推一年轮椅。</p> 
<p>两个人都不怎么困难地考取了美国的同一所高等学府，又未曾多加准备就飞过去开始了大学生活，十九岁的承太郎原本已经完全习惯了生活被挚友所填满，每周某个固定的午后要带花京院去医院复诊这种事就像是每天睡前要洗漱一样平常，互相占据对方的闲暇时光也再自然不过，本以为大学以后也会是这样，又或许，或许这一辈子都会是这样吧，然而学期开始了三周之后，花京院和他见面的时候却带来了第三个人。</p> 
<p>空条承太郎看待几乎任何女生，一眼扫过去基本都能自动扫描成一排排土豆，只是那天那个女孩子站在花京院身边，他才屈尊纡贵地多看了几眼，然而哪怕是这样，事到如今也早已不记得那姑娘的高矮胖瘦美丑与否，只记得她和他们一样是日本到美国来的留学生，是黑头发，她在见到承太郎的时候伸过手来，落落大方地说：</p> 
<p>“你好，我是典明的女朋友。”</p> 
<p>她叫他典明，而承太郎不怎么情愿地在那只很小的手上稍微握了握就松开了。</p> 
<p>三个人一起吃了顿饭，席间承太郎很快发现花京院的目光一直都在往自己这边看，抬头对上眼镜片后面那双熟悉的眼睛之后，对方却又很快移开了视线。承太郎揣测好友大概是想要自己的一两句评价或是祝福吧，因此他咽下一口食物，开口干巴巴地说：“你们俩挺配啊。”</p> 
<p>那姑娘似乎是说了两句“讨厌啦哪有”之类的推辞，承太郎没加注意，他紧紧盯牢花京院直到对方终于回过头来对他笑笑，说，谢谢你承太郎。</p> 
<p>承太郎点点头，继续吃着盘里的土豆泥，咀嚼的过程中颇感食不下咽，承太郎心想，美国菜真的好难吃。</p> 
<p>这顿饭吃的算不上愉快，不过花京院早就知道承太郎和任何陌生人吃饭都不会特别愉快，因此他宽宏大量地没有见怪，饭后承太郎结了账站起身来，习惯成自然地开口，我送你回宿舍吧，花京院回过头挑挑眉头说，那谁送她回去？</p> 
<p>“让她自己回宿舍好了，有什么关系，现在天都没黑。”</p> 
<p>这句话在舌头上打了个转还是没说出来，承太郎有些尴尬地把已经搭在了花京院肩膀上的手收了回来，点点头说好，又没话找话地加了一句那你路上小心，花京院当然还是说了谢谢。认识这些年以来承太郎早就知道花京院习惯对任何人的善意时刻道谢，但是此时的这声谢谢却让他充满了隔阂感，就好像一堵无形的空气墙凭空竖起来说，谢谢，承太郎，够了，你不要再过来了。</p> 
<p>他傻呆呆地站在餐桌边目送那对情侣慢悠悠离开的背影，看了好一会儿之后心想，至少，至少他们还没牵手。</p> 
<p>心里的另一个小声音说，好像他们在你看不见的地方就永远不会牵似的。</p> 
<p>往自己的宿舍走的时候，承太郎的心情不好，他不至于像十年之后认识的虹村家的男生一样，把朋友交了女朋友的事就当做不可饶恕的背叛，空条承太郎受欢迎了这么多年，他觉得自己不会因为花京院先交到个女朋友就嫉妒得胸口发紧。那么此刻这种不愉快的心情究竟来自于哪里呢，他回到宿舍就窝进了床，上大学以后第一次做出了吃完了就睡这么颓废的事，然而就算躺下来什么也不干，想了好久还是没能理清楚头绪。</p> 
<p>最后承太郎得出了结论，或许无论如何，对于花京院先交了女朋友的事他还是多多少少有点嫉妒的吧，人之常情，不必在意了。</p> 
<p>.</p> 
<p>大学三年级的寒假，两人一起回日本过新年，大一寒假的时候花京院曾经说过念书的地方那么远就不必每个假期都回国了，多浪费时间金钱啊，结果承太郎把他一把塞进私人飞机里带回了日本，之后的每个假期花京院也就不再抗议回家这件事了，只是飞机的规格从私人飞机变成了普通民航的头等舱。</p> 
<p>两年半的时间里花京院陆陆续续地又换过几个女朋友，日本的留学生也有，美国本地的女生也有，最长也只不过在一起五个月。刚开始他还把每个姑娘都带过来给承太郎过目，后来再开始新恋情的时候索性就只是在对话中随口一提，而还没等承太郎在学校里找到那个姑娘的影子，花京院又在某次随口一提的时候说，哦她啊，已经分了。</p> 
<p>飞机上两个人蜷在头等舱的座位里，昏昏沉沉地想把回程的十多个小时睡过去，结果睡了没多久却在某个时刻同时醒了过来。</p> 
<p>承太郎把头搁在花京院肩膀上混混沌沌地问了一句现在几点了，花京院稍微在座位里坐坐直，好让承太郎靠过来的时候不至于脖子太累，他抬起身边的窗挡向外看去，云层尽是一片昏昏沉沉的暗蓝色，视线的尽头处少许染上了些微紫红，花京院回头轻轻说，不知道在哪个时区，不过应该是凌晨吧，反正不管几点都能继续睡。</p> 
<p>承太郎沉默一会儿靠在他肩膀上叫了一声他的名字。</p> 
<p>“……花京院。”</p> 
<p>“嗯？怎么了？”</p> 
<p>他停顿一会儿，用刚睡醒的昏昏沉沉的大脑思考着自己的这个问题问出来是否合适，心中的某个部分催促着自己说句“没什么”应付过去算了，另一个部分却理性地分析着“你想问他这件事很久了，别拖了，问吧。”</p> 
<p>花京院见他没有动静，就说你想好了再告诉我吧，要是忘了就继续睡，以后会想起来的。</p> 
<p>承太郎心想我应该是想好了，于是他清了清嗓子问：“呃，花京院，你是打算继续念书？还是毕业了直接去工作？”</p> 
<p>他靠在花京院的肩膀上，整个脸都凑在他脖子旁边，因此很明显地能感觉到对方迟疑了一下。花京院过了好一会儿才说：“这个，我还没想好呢。”</p> 
<p>光是对方没有因为他管得太多而显示出不悦这点，承太郎就已经非常满足了，听到回答之后他立刻说，你不用马上想好，我就是随口一问，别在意。</p> 
<p>“不，这个事我的确该考虑起来了，承太郎你呢？”</p> 
<p>“我会去M大学继续念现在这个专业的PhD。”</p> 
<p>没有“我想要”，“我大概”，直接就是“我会去”，这样的遣词的确是承太郎的风格，花京院听到之后笑了笑。</p> 
<p>“其实这个事我也考虑过一阵子，就是没下定决心而已，”他伸手把承太郎额头前睡乱了的头发向后拨，拇指顺着耳朵的弧线向后滑到脖子边的皮肤，“这样吧，我寒假前，不，寒假结束前一定给你答复，好吗？”</p> 
<p>“……好。”</p> 
<p>承太郎往花京院身上拱了拱然后就再次闭上了眼睛。其实花京院完全没有义务向他报告自己的决定，更没有必要限定一个时间截点给承太郎答复，但是他没有说穿，花京院主动这样说让他觉得很开心，让他觉得几年来感受到过那么几次的那种空气墙终于轰然倒塌，而他和花京院之间也终于恢复了那种堪称完美的距离。这种开心没法跟任何人说，承太郎只是在心里的某个角落暗暗地高兴着，连嘴角都控制不住地要往上翘。</p> 
<p>感觉，稍微有点像他第一次听到花京院说和女朋友分手了的心情。</p> 
<p>.</p> 
<p>承太郎在除夕的夜晚接到了电话，听筒里传来和自己身边同样的电视节目背景音，他“喂喂？”了好几声才从歌声中分辨出对方说的话。</p> 
<p>“明早去参拜吗？”</p> 
<p>他回头问了一声母亲，荷莉点头说当然可以，承太郎于是对话筒说可以，约个时间在花京院家前面第一个路口见面好了。对方少许沉默了一阵，正当承太郎心想是不是应该把自己这边的电视机关掉的时候，花京院再次开口：</p> 
<p>“承太郎，回来的飞机上问我那个事，还记得吗？”</p> 
<p>他心里咯噔一下，接着急急匆匆地说当然记得。</p> 
<p>“明天会给你答复的。”</p> 
<p>“诶，已经想好了？”</p> 
<p>“是啊，总算，”将这件事说出口的花京院很明显地松了口气，从喘气声来听，大概是在电话那头笑了起来吧，“那么明天见。”</p> 
<p>“等等！”承太郎在对方还没有挂断电话的时候对着话筒大叫了一声，“反正你都想好了，现在就告诉我好了！”</p> 
<p>听筒中传来的回答这次毫不犹疑：“不行。”</p> 
<p>“为啥？！”</p> 
<p>“晚安承太郎，做个好梦。”</p> 
<p>花京院当机立断地挂断了电话，留下承太郎对着一串嘟嘟嘟的忙音不知该作何反应。母亲招呼他坐下来吃剥好了的橘子，他浑浑噩噩地走过去坐下把甜甜的水果往嘴里塞。母子二人看着红白歌会的这个时刻，世界的其他地方自然也有着不同风格的新年音乐会，空条贞夫因此也一如既往地不见踪影。母亲问他刚才打电话来的是谁，他如实回答说是花京院，荷莉笑着说，怎么还是花京院君呀。</p> 
<p>承太郎咬着橘子没有回答，荷莉于是也习惯性地自说自话起来，哎呀，看承太郎这么激动，我还以为是女孩子打来的电话呢。</p> 
<p>他想说我才没有激动呢，但是刚才对着电话听筒大叫的声音似乎还没从耳朵边上散开，因此只是闷闷地继续嚼着水果不再辩解。荷莉大概早就习惯了儿子一直以来就是这副什么都憋着不说出口的样子，没得到应答也不失落，继续高高兴兴地看着电视节目，手里一刻不停地把小小的剥好的橘子往承太郎手里送。</p> 
<p>这个除夕之夜直到他关掉电视跟母亲说了晚安又钻进自己的被窝都一直过得云里雾里，承太郎觉得思考什么都困难，无论想到什么事，大脑都迂回地绕着弯把思绪往同一件事上面带。他把脸埋进枕头心想反正明天就知道了，现在纠结又有什么用，睡觉睡觉。</p> 
<p>然后这一夜他都没有睡着。</p> 
<p>.</p> 
<p>新年第一天的清晨，空条承太郎顶着黑眼圈出了门。</p> 
<p>躺在被窝里很困却怎么都睡不着并不是什么愉快的体验，虽然距离答案揭晓已经越来越近，而且这个答案也不会因为他的意志而改变，但是承太郎还是忍不住一遍遍地想，要是花京院留下来继续念书就好了，要是花京院继续待在我身边就好了，花京院可千万别选择回日本工作啊千万别千万别，我不想跟他分开，别走别走别走。</p> 
<p>他就这样在脑子里转着这些念头，听着房间里的座钟滴滴答答分分秒秒地走过去，直到最后天都亮了，承太郎揉揉眼睛坐起身来穿衣服，心跳得比当年查SAT成绩的时候还快。</p> 
<p>直到这时候，他依旧觉得会有这些想法只是因为自己不想让好朋友离自己太远。</p> 
<p>夜晚的臆想比起白天的会稍微乐观点，午夜时毫无道理地觉得花京院会留下来继续在美国念书甚至会和他选择同一个学校，起床时已经觉得事情没那么容易，不过如果花京院回日本工作的话自己还是能时不时坐飞机回来找他的，到了他裹好了衣服围巾出门走上了马路，承太郎的心开始悬起来了。</p> 
<p>他是挺败家，但是没有败家到会三天两头联系SPW弄飞机回国就为了去见一个朋友，而他很清楚自己对见不到花京院的时间的最大忍耐限度不会超过三天两头。</p> 
<p>怀着这样的心情走到约定的地点，花京院已经站在那里了。几个小时前承太郎曾经决定一见到他就要立刻问出来他的决定究竟如何，事到临头又害怕起来，两人打了个招呼之后就再没说话，气氛稍显尴尬，并肩走了一会儿之后，承太郎伸手把脖子上的围巾解了下来。</p> 
<p>他拍拍花京院的肩膀让他停下脚步，伸手解开了对方脖子上松松垮垮还很薄的围巾，转而换上自己的，花京院没有推拒，他眼睛看着地面，鼻头因为冷空气而冻得红红的，在承太郎打好了结之后嘟囔了一句谢谢，承太郎把花京院的围巾往自己脖子上随手一绕。</p> 
<p>“天这么冷你围着这个就跑出来了，新年就想弄感冒？”</p> 
<p>“我听预报说今天好像没那么冷。”</p> 
<p>“你就听了个最高气温吧。”</p> 
<p>以此为契机开始了闲聊，他们都小心翼翼地没有触及到昨晚说过的那个话题，直到随着神社的人龙步步前进，参拜过后又投了香火钱。承太郎许愿的时候心想今年的愿望还是老样子吧，老妈老头花京院啥的，反正就是这拨人身体健康别出意外，开开心心的，还有世界和平，谢谢了。</p> 
<p>睁开眼睛正准备离开，往边上一看，花京院依旧紧紧闭着眼，承太郎也就站在一边等他。</p> 
<p>花京院睁开眼睛之后，两人都没有说话，去抽了签之后互望一眼，都是小吉，再把签纸往外套兜里随手一揣就逆着人流的方向往回走了，路上不时有盛装打扮的姑娘停下来对承太郎多看几眼，承太郎看着那些五颜六色的和服料子从眼前飘过，觉得心烦得不行。</p> 
<p>走到了人稍微少些的地方之后他开始沉不住气了。你说好的把考虑的结果告诉我的呢，快说啊，别吊着我了，再不说都该各回各家了。但是催促的话在嘴里打了个转最后还是咽下去，承太郎最终开口的时候说的是，你刚才许愿许什么了？</p> 
<p>花京院没有看他，低头走路但是放慢了步子，沉默了挺久之后说：</p> 
<p>“希望明年申请M大学的研究生能成功。”</p> 
<p>承太郎停下了脚步，花京院于是也跟着停了下来。他们俩沉默着看向对方，花京院的下半张脸都窝在承太郎那又厚又大的围巾里了，看不太出他的表情到底怎么样。承太郎停了一会儿说：“M大学？”</p> 
<p>“对，就你要去念博士的那个大学。”</p> 
<p>对方说完之后挑了挑眉头，就好像在挑衅地说“怎样？”，承太郎还是愣在原地。这种感觉就好像你在心里埋了一颗未爆弹，担惊受怕了好久它会把你炸得体无完肤鲜血淋漓，结果等有人真的踩到它的时候，它砰地一声炸了，炸开来之后你才发现，他妈的原来这是一颗礼花弹啊。</p> 
<p>“其实也犯不着特意许愿，我个人觉得申请应该没什么问题，”花京院看他没有回答就继续说了下去，“毕竟我只是去念个硕——承太郎？！！”</p> 
<p>承太郎冲过去一把抱住了他，隔着新年里臃肿的衣物死命把胳膊往花京院身上挤，抱住了之后又觉得这个动作实在不足以表达自己有多高兴，怀里的花京院被一层层衣服裹得都找不到腰了，但是稍微掂量一下还是能感觉出来，他和高中的时候一样轻，轻得足以承太郎紧紧抱住他之后能继续提起来转两个圈。</p> 
<p>承太郎于是从善如流抱住花京院转了一圈，对方在离地的时候吓得一把搂住了他的脖子嘴里惊呼你干嘛呢，第二圈的时候花京院也开始笑了，他们俩狂笑着这样转了四圈半，如果不是承太郎由于各种原因开始晕的话或许还会再多转几圈。太阳穴都突突跳着的承太郎把花京院放下地，早上还只是鼻尖被冻红了的花京院此刻耳朵和脸颊上也都泛起了一片粉红，就像大福团子里的草莓透过薄薄的一层糯米糍晕出了颜色，他们一时之间谁都没有说话，呼吸都有些急促，笑着看进对方的眼睛。</p> 
<p>我好想亲他。</p> 
<p>这个念头像一道春雷劈进了脑子，震得承太郎自己都一惊，可是冲动来得太快太猛烈来不及等他在心里想明白到底是怎么回事。花京院还站在对面咧着嘴傻笑，承太郎已经把脸凑了过去，感觉此时去吻花京院似乎是一件拥抱着转圈大笑之后水到渠成自然而然的事。他闻到对方身上那股衣物柔顺剂味道钻进鼻子，平平淡淡的香味，但是撩得他心里发痒，嘴唇一寸寸凑近，眼看着快要成功的时候那人却正巧转过了头。</p> 
<p>花京院扭过头笑着说，人行道正中间这样像什么样子啊。</p> 
<p>承太郎知道他这句话不是针对自己刚才突如其来的接吻冲动，而是两个人刚才堵在人行道正中间又笑又闹的事，这句话迅速把他打回了现实，于是他把夭折的吻变成了另一个深深的拥抱。</p> 
<p>在人来人往的路上不顾一切地想要吻他，这事儿大概是有点过了，但如果只是抱抱应该并没有什么问题。</p> 
<p>花京院在他怀里安静了一会儿，细密的鼻息透过他自己那条松松垮垮的围巾落在承太郎的脖子上，这样安静了足有半分钟之后他开口了：</p> 
<p>“好啦，高兴够了吧，快放手。”</p> 
<p>“……嗯。”</p> 
<p>松开的胳膊多少有些恋恋不舍不情不愿，但是承太郎自己也知道不能再多做什么了，花京院从他胸前挪开，贴着他走在身边，两人的胳膊在行走时不时挨挨碰碰，有一搭没一搭地说着要去M大学念硕士或是博士的话得在这一年里都准备些什么东西，他们交流得很平静，仿佛刚才爆发的那一瞬间只是意外。</p> 
<p>承太郎嘴里跟着花京院的话题走，心却在胸腔内搅成一团。</p> 
<p>完了。</p> 
<p>我喜欢上花京院了。</p> 
<p>这是二十一岁的空条承太郎的单恋正式开始的一瞬间。</p>
